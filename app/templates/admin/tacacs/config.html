Gaia recipe
  1. Gaia Portal → Authentication → TACACS+: add server = this host, port 49, secret above.
  2. Map a role (the simulator returns priv-lvl=15 so admins get full access).
  3. SSH/login to Gaia as a demo user below with their directory password.
  4. ASCII and PAP logins are supported; watch them in the live log.
